PhotoManager API Specification. This overview gives some hints to get the orientation in the the program. Primarily an introduction to the program structure. Therefore it is strongly recommended to read this overview before you lost is the jungle of code.

General Information

All the pictures in and under the top level picture directory are managed by the program. The gui is realized with pure java swing. All are done by hand, nothing is done with a gui builder. Apaches lucene is the data base.
